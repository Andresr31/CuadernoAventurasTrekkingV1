# 🏔️ Cuaderno de Aventuras Trekking — Cumbres de Santiago

Una guía interactiva y visualmente impactante de las 12 cumbres más emblemáticas accesibles desde Santiago de Chile.

## 🌐 Ver en vivo

Desplegado en GitHub Pages: [Ver sitio](https://tu-usuario.github.io/CuadernoDeAventurasTrekking/)

## 📋 Contenido

- **12 fichas de cumbres** con datos técnicos verificados (altitud, desnivel, distancia, duración, dificultad)
- **Rutas detalladas** con estado actual de accesos
- **Manual de equipamiento** por categoría
- **Guía por estación del año** con recomendaciones específicas
- **Widget de clima** con datos de junio 2026
- **Frases motivantes** en carrusel interactivo
- **Números de emergencia** de montaña
- **Navegación interactiva** con scroll reveal, progress bar y back-to-top

## 🎨 Diseño

Estética editorial tipo "cuaderno de campo" con:
- Tipografías: Fraunces (display), EB Garamond (body), JetBrains Mono (datos técnicos)
- Paleta terrosa: papel envejecido, verde musgo, ocre, óxido
- Textura de grano y gradientes atmosféricos
- Animaciones de scroll y micro-interacciones
- Diseño responsive completo

## 🚀 Despliegue en GitHub Pages

1. Crear un repositorio en GitHub llamado `CuadernoDeAventurasTrekking`
2. Subir el contenido de esta carpeta
3. En Settings → Pages → seleccionar rama `main` y carpeta `/ (root)`
4. El sitio estará disponible en `https://tu-usuario.github.io/CuadernoDeAventurasTrekking/`
